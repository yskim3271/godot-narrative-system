# 통합 데모 (Integrated Demo)

Narrative System의 모든 MVP 기능을 한 씬에서 시연합니다. 프로젝트를 열고 ▶(메인 씬 실행)을 누르면 바로 시작됩니다.

## 조작

| 키 | 동작 |
|---|---|
| 방향키 | 이동 |
| **E** | 경비병 근처에서 대화 시작 |
| Enter/Space/클릭 | 대사 넘기기 (타자기 효과 중에는 즉시 완성) |
| **J** | 퀘스트 로그 열기/닫기 |
| **K** | 한국어 ↔ English 전환 (대화 중에도 즉시 반영) |
| **F5 / F9** | 저장 / 불러오기 (선택지 화면 한가운데서도 동작) |

## 체험 순서 (모든 기능 확인)

1. **분기 대화 + 조건**: 경비병(빨간 사각형)에게 다가가 E — 첫 대화에서 "처음 보는 얼굴이군"이 나옵니다. 대화를 끝내고 다시 걸면 **"또 자네군"** (seen-node 추적 기반 첫만남 분기).
2. **조건부/비활성 선택지**: "통행료를 내겠네 (50골드)"는 골드 30으로는 **회색 비활성**. "일거리가 있나?"는 퀘스트 시작 후 사라지고 진행/완료 보고 선택지로 바뀝니다(`objective_count` 조건).
3. **퀘스트 + 컷신**: "일거리가 있나?" 선택 → 퀘스트 시작 알림 + **시퀀서 컷신**(표정 변경 → 손 흔들기 애니메이션 → 카메라가 경비병에게 팬 → `quest_given` 시그널(콘솔 출력) → 카메라 복귀).
4. **퀘스트 트래커/로그**: 우상단 HUD에 "쥐 사냥 (0/5)". J로 로그를 열면 설명·목표·트래커 토글.
5. **objective 진행**: 회색 쥐 5마리를 밟으면 처치 알림 + 트래커 갱신. 5마리 후 경비병에게 "쥐를 모두 처리했네." → **완료 + 보상 100골드** → 골드 130으로 통행료 선택지가 활성화됩니다.
6. **저장/불러오기**: 아무 때나(선택지 표시 중 포함) F5 → 게임을 계속 진행 → F9 → 변수/퀘스트/대화 위치/쥐 상태까지 복원.
7. **로컬라이징**: K를 누르면 표시 중인 대사·선택지·퀘스트 UI·알림이 즉시 영어로.
8. **Bark**: 대화하지 않는 동안 경비병 머리 위에 4초마다 혼잣말 말풍선.

## 데이터 구조

- [demo_database.tres](demo_database.tres) — Inspector에서 열어 편집 가능한 데이터베이스 (캐릭터/대화/퀘스트/변수/번역 테이블)
- [db_builder.gd](db_builder.gd) — 위 .tres의 원본 (코드 저작). 수정 후 재생성:
  ```
  godot --headless --path . -s res://examples/integrated_demo/regenerate_db.gd
  ```
- 이 데이터베이스는 테스트 스위트(`test_demo_database.gd`)가 항상 validator로 검사하므로 깨진 채 커밋될 수 없습니다.
