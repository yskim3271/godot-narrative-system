# 대화 그래프 에디터

에디터 상단의 **Narrative** 메인 스크린 탭(2D/3D/Script 옆)에서 대화를 노드 그래프로 보고 편집합니다. 데이터베이스는 `narrative_system/database_path` 프로젝트 설정에서 읽습니다(하단 Narrative 패널에서 Load하면 자동 설정).

## 화면 구성

- **툴바**: 대화 선택 드롭다운 · New Dialogue · Add Node · Set Start · Delete · **1.2.3**(선택지 넘버링) · Save · Validate · Refresh · 상태 표시
- **캔버스(GraphEdit)**: 줌/팬/미니맵 기본 지원. 노드마다 —
  - 제목 = 노드 id 읽기 전용 표시 (시작 노드는 `▶ ` 접두)
  - 헤더 행: **id 입력칸**(rename) · **화자 입력칸** · 배지 (❓조건 ⚡액션 🎬시퀀스) — 왼쪽 입력 포트 / 오른쪽 **next** 포트(파랑)
  - **텍스트 편집칸**(여러 줄)
  - 선택지 행들: **선택지 텍스트 입력칸** · **타깃 id 입력칸**(`(end)` = 종료) — 오른쪽 **choice** 포트(노랑), 선택지당 1개

## 조작

| 작업 | 방법 |
|---|---|
| 노드 추가 | 캔버스 우클릭 → *Add Node Here* (또는 툴바 Add Node = 화면 중앙) |
| 연결 | 출력 포트(파랑=next, 노랑=choice)에서 대상 노드의 입력 포트로 드래그 — `next_node_id`/`target_node_id`가 갱신되고, 같은 포트에서 다시 드래그하면 교체 |
| 연결 해제 | 연결을 입력 포트에서 떼어내기(`right_disconnects`) |
| 노드 삭제 | 선택 후 Del 또는 툴바 Delete — **해당 노드를 가리키던 모든 링크 자동 정리** |
| 시작 노드 지정 | 노드 1개 선택 → Set Start |
| 텍스트/화자 인라인 편집 | 노드의 화자·텍스트 칸에 직접 입력 → 포커스를 떠나면 1회 편집이 1 undo 단위로 커밋 |
| 선택지 인라인 편집 | 선택지 행의 **텍스트 칸**과 **타깃 id 칸**에 직접 입력. 타깃을 비우면 "대화 종료", 없는 노드 id는 거부 후 되돌림. 포트 드래그로 연결을 바꾸면 타깃 칸도 따라 갱신됩니다 |
| 노드 id 변경(rename) | 헤더의 id 칸에 입력 → Enter/포커스 이동 시 적용. **그 id를 가리키던 next/choice 링크와 start_node_id를 모두 자동 추적**(끊긴 링크가 생기지 않음). 잘못된/중복 id는 거부되고 칸이 되돌려집니다 |
| 마크업 삽입 | 텍스트/선택지 텍스트 칸 안에서 — **Ctrl+Shift+V** = `[var=…]` 삽입(선택 영역이 변수명이 됨), **Ctrl+Shift+C** = 선택 영역을 `[color=…][/color]`로 감싸기. → [dialogue_authoring.md](dialogue_authoring.md)의 인라인 마크업 |
| 선택지 자동 넘버링 | 노드 1개 선택 → 툴바 **1.2.3** 또는 **Ctrl+Shift+N** — 선택지 텍스트에 `1. `, `2. ` 접두를 토글(한 번 더 누르면 제거, 1 undo 단위) |
| 그 외 필드 편집 | 노드 클릭 → **Inspector**에서 조건/액션/시퀀스·선택지 조건 등 편집 (❓⚡🎬 배지로 표시) |
| 배치 저장 | 노드를 끌어 정리 → Save (위치는 노드 `metadata.graph_position`에 .tres로 저장) |
| 실행 취소/복구 | 위 모든 편집·구조 변경은 에디터 표준 **Ctrl+Z / Ctrl+Y**로 취소·복구 |
| 새 대화 | New Dialogue → id 입력 (start 노드 자동 생성) |

처음 여는 대화는 시작 노드 기준 BFS **자동 배치**됩니다(깊이=열, 분기=행).

## Inspector와의 관계

그래프와 Inspector는 **같은 리소스 인스턴스**를 편집합니다. Inspector에서 선택지를 추가/삭제하는 등 구조를 바꿨다면 **Refresh**(또는 탭 재진입)로 캔버스를 다시 그리세요 — 위치·연결은 데이터에서 재구성되므로 잃지 않습니다.

## 현재 한계

- 인라인 편집은 화자/텍스트/노드 id/선택지 텍스트·타깃에 한정 — 조건/액션/시퀀스·선택지 조건과 선택지 **추가/삭제**는 Inspector(구조 변경 후 Refresh)
- 노드 id를 *이미 끊긴 링크가 가리키던 id*로 rename하면 그 링크가 새 노드에 연결됩니다(극히 드문 경우, undo 비대칭). 평소엔 Validate가 끊긴 링크를 잡아줍니다

## 검증과의 연동

툴바 **Validate**는 요약(에러/경고 수)을 상태줄에 표시합니다 — 상세 목록은 하단 Narrative 패널의 Validation 탭에서 확인하세요.
